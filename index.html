Kéo thả

Lắp truy vấn SELECT cơ bản rồi chạy trên chính database đang mở.

  • Tạo database: dùng nút “Tạo database”; SQLite không có câu lệnh `CREATE DATABASE`.
  • Lưu dữ liệu: thay đổi được tự lưu vào IndexedDB sau khi transaction kết thúc.
  • Sao lưu: dùng “Xuất file” để tải database chuẩn SQLite về máy.
